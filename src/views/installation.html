Style Guide Installation

1. Download the Style Guide

First things first, download the contents of the Github repo.

2. Make sure you have node.js and npm installed

You can download node.js from the node.js site.

3. Start the development environment

In terminal, navigate to the your downloaded style guide folder, then use the following command to start the development environment:

$ npm start

This installs dependencies, compiles the site, and watches for changes to re-compile.

4. Make your changes and edits

Make your changes to the files inside /src.

/src/assets/toolkit/styles/ contains the .scss files with all of the button, typography, etc styles.

/src/materials/ contains the individual component pages with code and descriptions.

5. Run an optimized build

Once you're happy with your changes and you're ready to deploy, run the following command to run an optimized build

$ npm run build
